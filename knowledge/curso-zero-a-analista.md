# Do Zero a Analista - Formação Completa em Análise de Dados

**Instrutor:** Caio Gomes (mesmo do CDO)

## Sobre a Formação

- **Foco:** Formar Data Analysts e Cientistas de Dados Jr. Cobre desde fundamentos até skills avançados que o mercado realmente demanda, não o "básico" que todo curso ensina.
- **Volume:** ~427 aulas organizadas em 14 módulos, cobrindo Excel, SQL, Python, estatística, inferência, modelagem, visualização, comunicação e preparação para o mercado.
- **Formato:** Vídeos de 5-15 minutos, cada um ensina um único conceito. Sem aulas de 1 hora com slides intermináveis. O aluno escolhe a profundidade: entendeu rápido, pula pro próximo; precisa de mais, os vídeos estão lá.
- **Profundidade estatística real (diferencial):** Inclui Inferência Estatística completa (75 aulas) e Modelagem Estatística completa (90 aulas): OLS, regressão logística, GLM, GEE, bootstrap, testes de hipótese, correções para múltiplas comparações. Forma profissionais que sabem pensar estatisticamente, não apenas operar ferramentas.
- **Data cleaning como skill central:** Cursos de mercado usam dados perfeitos. Aqui: células mescladas, datas em 5 formatos, encoding errado. É o que analistas realmente enfrentam e onde gastam 40-50% do tempo.
- **Business context + preparação de carreira:** Módulo de Cases por Vertical (e-commerce, fintech, varejo, marketing, SaaS) e módulo de Carreira em Dados (CV, entrevistas técnicas, take-homes, negociação). O que bootcamps de $15K cobram, incluído no preço base.
- **Acesso:** Assinatura anual. Enquanto ativa, acesso a todo conteúdo incluindo atualizações e novos módulos.
- **Garantia:** 7 dias. Reembolso integral se não fizer sentido.
- **Comunidade:** Discord para tirar dúvidas, compartilhar projetos e interagir com outros estudantes e profissionais.
- **Aulas gravadas:** Editadas para ser curtas e diretas. Assiste no seu ritmo.
- **Funciona em qualquer dispositivo:** Computador, tablet, celular.
- **Sem prazo:** 427 aulas de 5-15 min cada. Dedicando 1 hora por dia, dá pra completar em alguns meses. Mas não existe prazo.

## Público-alvo

Para quem quer entrar na área de dados partindo do zero: recém-formado, profissional em transição de carreira, ou alguém que já trabalha com dados mas nunca teve formação estruturada. O curso assume zero conhecimento prévio de programação, estatística ou ferramentas. Não precisa saber programar (ensina Python do zero). Não precisa ser bom em matemática (conceitos estatísticos construídos passo a passo com exemplos visuais).

## Tópicos da Formação (~90 tópicos em 14 módulos)

### 1. A Profissão de Data Analyst
O que um DA faz de verdade, as 5 etapas (Coletar, Limpar, Analisar, Visualizar, Comunicar), skills técnicos vs soft skills, path de carreira (DA, Senior, Lead, Manager/Scientist).

### 2. Excel Avançado
PivotTables e análise exploratória, fórmulas avançadas (XLOOKUP, INDEX/MATCH, arrays dinâmicos), Power Query (ETL no Excel), boas práticas e armadilhas.

### 3. SQL
Fundamentos (SELECT, WHERE, ORDER BY, GROUP BY), JOINs (INNER, LEFT, RIGHT, FULL, self-joins), subqueries e CTEs, Window Functions (ROW_NUMBER, RANK, LAG, LEAD), performance e otimização (EXPLAIN, índices), DuckDB para analytics moderno.

### 4. Python para Análise
Fundamentos Python, pandas (GroupBy, merge, pivot), data cleaning na prática (missing values, duplicatas, tipos, strings, datas), visualização (Matplotlib, Seaborn), automação de tarefas.

### 5. Estatística Aplicada
Tipos de variáveis, medidas de tendência central e dispersão, distribuições (normal, binomial, Poisson), visualização estatística, correlação (Pearson, Spearman, e por que correlação não é causalidade).

### 6. Probabilidade
Conceitos fundamentais, probabilidade condicional, Teorema de Bayes, variáveis aleatórias, distribuições de probabilidade.

### 7. Inferência Estatística (75 aulas)
Amostragem e vieses, intervalos de confiança, distribuição t de Student, testes de hipótese (nula, alternativa, erros Tipo I e II), p-valor, poder estatístico e tamanho de amostra, múltiplas comparações (Bonferroni, Benjamini-Hochberg), bootstrap e reamostragem.

### 8. Modelagem Estatística (90 aulas)
Correlação de Pearson e Spearman, regressão linear simples (OLS, R2, resíduos), regressão linear múltipla (multicolinearidade), ANOVA, Qui-Quadrado, regressão logística (odds ratio), GLM (Gaussiana, Binomial, Poisson como casos particulares), GEE (medidas repetidas, dados longitudinais).

### 9. Visualização e BI
Princípios de visualização, escolha de gráficos, Looker Studio (gratuito, web-based), Metabase (open source, SQL nativo), dashboards efetivos (métricas vs KPIs, drill-down).

### 10. Comunicação de Dados
Pirâmide invertida (conclusão primeiro, SCQA), data storytelling, escrita técnica para não-técnicos, apresentações executivas, respondendo ad hoc requests.

### 11. Cases por Vertical
E-commerce (CAC, LTV, cohorts, GMV vs receita), fintech (PDD, default rate, vintage analysis, risco), varejo físico (SSS, giro, ruptura, margem), marketing/growth (CAC, ROAS, funil, retention), SaaS/tech (MRR, ARR, NRR, churn, LTV/CAC).

### 12. Carreira em Dados
CV e LinkedIn para dados (ATS, portfolio, GitHub), entrevistas técnicas (estrutura típica, o que cada etapa avalia), SQL em entrevistas (10 padrões de pergunta, live coding), take-home assignments, negociação e primeiros 90 dias.

### 13. ML Básico
O que é ML (classificação, regressão, clustering), regressão linear na prática, classificação básica (regressão logística, métricas), validação e overfitting (train/test split, cross-validation).

### 14. Agentes de IA
MCP (Model Context Protocol), A2A (Agent-to-Agent communication), projeto Pesquisador Pessoal (agente de IA funcional do zero).

## Diferença para o "De Analista a CDO"

Este curso foca em formar Data Analysts e Cientistas de Dados Jr, cobrindo as ferramentas e a estatística que o mercado exige. O "De Analista a CDO" é uma formação contínua mais ampla, que inclui todo este conteúdo MAIS matemática avançada (álgebra linear, cálculo), inferência bayesiana e causal, séries temporais, e trilhas de liderança. Se o objetivo é ser analista ou DS Jr, este curso é suficiente. Se quer ir além ou já é mais experiente, o CDO pode fazer mais sentido. Quem compra o CDO tem acesso ao conteúdo do Zero a Analista também.

## Empregabilidade

O curso inclui módulo completo de Carreira em Dados: como montar CV, como se preparar para entrevistas técnicas, como fazer take-home assignments, como negociar salário. Mas emprego depende de muitos fatores. O curso dá as ferramentas e o conhecimento, a execução é do aluno.
