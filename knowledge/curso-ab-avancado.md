# A/B Testing Avançado

- **Público:** Profissionais com base em A/B testing que querem ir além do básico
- **Formato:** 3 aulas de 2h (ao vivo)
- **Conteúdo:** A/B testing avançado — inferência causal, CUPED, métodos sequenciais, bandit algorithms, interferência entre grupos
- **Diferenciais:** Não é o básico de "calcule o p-value". É o que as big techs usam de verdade
- **Pré-requisito:** Experiência prévia com A/B testing
